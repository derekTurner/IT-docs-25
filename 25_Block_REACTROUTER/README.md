# Full stack application using the remix framework

The best introduction to react router is the [Adress Book tutorial](https://reactrouter.com/tutorials/address-book#setup) on the framework's website.

A mongo database can be generated locally to run in a container or as a free atlas cluster.  A local installation is preferred if you are behind a firewall that may block access to the atlas cluster.

A library CRUD application will be developed as an example.
